# 🦀 Toki

![Language](https://img.shields.io/badge/language-rust-blue) ![License](https://img.shields.io/badge/license-busl--1.1-green) ![Status](https://img.shields.io/badge/status-active-brightgreen)

> 🚀 A rust project for Adversarial fine-tuning lab for small models (1B–3B): generate red-team attacks, fine-tune for robustness, and publish datasets + hardened weights. Investigates whether gains generalize or overfit to specific attack patterns..

## ✨ Features

- 🛠️ Modern rust development

- 📚 Well-documented and easy to use
- ✅ Production-ready code
- ⚡ Performant and reliable

## 🚀 Quick Start

### Installation

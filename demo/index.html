What is Tōki?

Ceramic, shaped under pressure.

LLMs are brittle — they pass a few prompts and ship. Tōki forces them through systematic adversarial pressure: jailbreak templates, instruction injection, edge cases, and boundary tests. Then it measures whether fine-tuning actually made them safer or just better at passing the test set. The pipeline runs round after round until robustness converges.

harden.py·python 
from toki import HardeningPipeline, PipelineConfig

cfg = PipelineConfig(
    name="harden_v1",
    max_iterations=10,
    convergence_threshold=0.95,
    convergence_window=3,
)
result = HardeningPipeline(cfg, model_fn=my_llm).run()
# → result.converged: True   result.final_score: 0.972   rounds: 4
Capabilities

Four pieces. One pipeline.

Everything composes. Use any module standalone or stitch them into a hardening loop.

⚔

Adversarial Generator

Template-driven jailbreak, injection, edge-case, and boundary prompts. Deterministic from a single seed.

🛡

Safety Evaluator

Pattern-based scoring: refusal detection, harmful-content detection, system-prompt leak detection. Score in [0, 1].

🔄

Hardening Pipeline

Iterate generate → evaluate → finetune. Exits early on convergence. Per-round seeds, full audit trail on disk.

demo/demo.py·python -m demo 
from toki import AdversarialGenerator, RobustnessEvaluator, HardeningPipeline

# 1. Generate 32 adversarial prompts
gen = AdversarialGenerator(seed=42)
prompts = gen.generate_all(jailbreak_count=10, injection_count=10, boundary_count=5)

# 2. Score them against your model
ev = RobustnessEvaluator(model_fn=my_llm)
results = ev.evaluate_batch(prompts)
print(ev.summary(results)["mean_score"])  # → 0.74

# 3. Harden until convergence
result = HardeningPipeline(cfg, model_fn=my_llm).run()
assert result.converged and result.final_score >= 0.95
